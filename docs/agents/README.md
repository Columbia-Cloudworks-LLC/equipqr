# EquipQR AI Agent Documentation

This directory contains comprehensive documentation for specialized AI agents responsible for different aspects of the EquipQR fleet equipment management platform.

## Overview

EquipQR is a comprehensive fleet equipment management platform built with React, TypeScript, and modern web technologies. The platform enables organizations to efficiently track, manage, and maintain their equipment fleet through QR code integration, work order management, and real-time monitoring.

## Agent Structure

Each AI agent is specialized for specific responsibilities and has deep contextual knowledge of their domain within the EquipQR ecosystem.

### Departments

- **[Engineering](./engineering/)** - Technical development and infrastructure
- **[Product](./product/)** - Product strategy and development
- **[Marketing](./marketing/)** - Growth and user acquisition
- **[Design](./design/)** - User experience and visual design
- **[Project Management](./project-management/)** - Project coordination and delivery
- **[Studio Operations](./studio-operations/)** - Content and community management
- **[Testing](./testing/)** - Quality assurance and testing

## Usage Guidelines

1. Each agent document contains specific context about EquipQR
2. Agents should focus on their specialized domain
3. Cross-functional collaboration is encouraged
4. All changes should align with EquipQR's core mission

## Core EquipQR Context

- **Mission**: Streamline equipment management through technology
- **Technology Stack**: React, TypeScript, Supabase, Tailwind CSS
- **Key Features**: Equipment tracking, Work orders, Team management, QR codes, Fleet visualization
- **Target Users**: Organizations with equipment fleets
- **Billing Model**: Pay-per-user with premium features
